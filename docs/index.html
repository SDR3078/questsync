A companion for your Habitica tasks

Tick your Habitica tasks from anywhere.

QuestSync surfaces your Habitica to-dos and dailies in any CalDAV app — Tasks.org, Apple Reminders, Thunderbird, Nextcloud Tasks. Read them, check them off, done. Habitica stays the single source of truth; QuestSync is stateless and stores nothing.

Getting set up

Three ticks to sync

Add a CalDAV account

In your calendar or tasks app, point a new CalDAV account at your QuestSync instance. Use your Habitica User ID as the username and your API token as the password.

See your to-dos & dailies

Your Habitica tasks appear as native items in Tasks.org, Apple Reminders, Thunderbird, Nextcloud Tasks — anywhere that speaks CalDAV.

Tick anywhere — Habitica stays canon

Check a task off on your phone and it lands back in Habitica. QuestSync keeps no copy; Habitica remains the single source of truth for your to-dos and dailies.

Open source · Free

Self-host it

Own the whole thing. Clone the repo and bring it up with one Docker command — it is open source and free forever.

git clone https://github.com/SDR3078/questsync
cd questsync
docker compose up -d --build

Then add a CalDAV account using your Habitica User ID as the username and your API token as the password.
